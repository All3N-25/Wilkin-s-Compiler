# 🛠️ Wilkin's Compiler

A mini compiler built in Python that supports lexical analysis, semantic analysis.

---

## 📦 Installation

Choose one of the following setup options:

### Option 1 — ZIP File
1. Download and unzip the submitted `.zip` file.
2. Navigate into the extracted folder.

### Option 2 — Clone via Git
```bash
git clone https://github.com/All3N-25/Wilkin-s-Compiler.git
```

---

## ▶️ How to Run

1. **Launch the editor** by running the main Python file:
   ```bash
   python WilkinsEditor.py
   ```

2. **Open or save a file** using the `File` menu — the compiler works with `.wil` source files.

3. **Compile your code** by pressing the `Compile` button — this saves and executes the current code.

4. **View the output** in the terminal window that appears after compilation.

---

## 📝 Notes

- Source files use the `.wil` extension.
- Make sure you have **Python 3** installed before running.
- The compiler will report any lexical or semantic errors directly in the terminal output.
