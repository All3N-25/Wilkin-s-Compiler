## How to Setup:
Option 1:
1. Unzip the submitted zip file.

Option 2:
1. Clone the github link.
```
git clone https://github.com/All3N-25/Wilkin-s-Compiler/tree/main
```

## How to run
1. Run the python file "WilkinsEditor.py".
2. You can press "file" to open and save a .wil file.
3. You can press "compile" to save and run the current lines of code.
4. A terminal window will appear, containing the program output.
